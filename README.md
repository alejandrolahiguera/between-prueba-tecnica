# Similar Products API

## Description
This project exposes a REST API to retrieve products similar to a given product.

## Requirements
- Java 21
- Gradle 8.0 or later
- Docker

## Build and Run

### Building the Application

1. **Clean and Install:**

   ```bash
   ./gradlew clean install

2. **Build:**

   ```bash
   ./gradlew bootJar

3. **Run from JAR:**

   ```bash
   java -jar build/libs/pruebatenica-0.0.1.jar
